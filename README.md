# LOGO LAND

## Description
**Logo Land** is a game where players must choose the correct logo of a brand or company as they progress. It includes a save system, a skin selector, an option to delete saved games, and a coin system. At the end of each game, it also displays the global coin record and the average response time per logo.

## Installation

### Prerequisites
- Python  
- Pygame  

### Installation Instructions
Clone the repository:
```bash
git clone https://github.com/Keruza3/Proyecto-python-juego
```
## Usage
### How to Play

To start the game:
Run the main.py file.
###Controls

Mouse: Navigate through menus and select options.

### Features

#### Save System: Saves up to 4 games with the player's name.

#### Skin Selector: Customize your character’s appearance.

#### Delete Saves: Remove saved games to free up space.

#### Coin System: Accumulate coins as you play.

#### Coin Record: Displays the highest number of coins accumulated by any user.

#### Average Response Time: Shows the average time you take to answer each logo.
